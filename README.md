# React E-commerce Platform

A modern e-commerce platform built with React, Vite, and TailwindCSS featuring both user and admin interfaces.

## Features

- 🛍️ User Interface
  - Product browsing and searching
  - Shopping cart functionality
  - User authentication
  - Responsive design
  - Dark/Light mode

- 👨‍💼 Admin Dashboard
  - Product management
  - User management
  - Order tracking
  - Payment monitoring
  - Analytics dashboard

## Tech Stack

- React 18.3
- Vite 6.0
- TailwindCSS 3.4
- React Router DOM 7.0
- React Icons
- ESLint
